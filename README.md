## What is this project?

**(EN)**

This is a web application for a luxury **dog-friendly beach resort**, developed using **React, TypeScript, and Tailwind CSS**.

Based on the concept of **"bringing your dog everywhere with you,"** this app showcases a resort where you and your furry friend can enjoy paradise together. With a smooth, elegant UI, immersive animations using Framer Motion, and interactive sliders powered by Swiper.js, the site offers a delightful user experience that mirrors a real high-end resort website.

The idea was **inspired by my own experience staying at a pet-welcoming hotel**. I thought, "Wouldn’t it be amazing if a place like this really existed?" That vision became the foundation for this project.

Designed with care for both desktop and mobile users, every detail—from responsive layout to animation—is fine-tuned for comfort and clarity.

In recent refactoring, I focused on performance improvements and maintainability. Enhancements include optimized responsive behavior, smoother slider performance, and a better mobile experience. UI elements like navigation and hover effects have also been polished to create a more intuitive and engaging feel.

Perfect for dog lovers dreaming of a luxury vacation with their four-legged companion! 🐶🏖️

**Fun fact**: One of the adorable dogs featured in the app is actually my own pup! 🐾
Can you spot him?

## 🐾 Features

**(EN)**

Here are the main features currently implemented in **Dogation**:

✅ **Home**

- A visual and welcoming introduction to the charm of a luxurious, pet-friendly hotel

✅ **Services**

- Features a carefully written overview of the resort’s offerings, **based on my own stay** at a real-life dog-friendly hotel
- Using **Framer Motion**, subtle yet engaging animations respond to user scroll, creating a smooth and memorable transition between sections.

✅ **Facilities**

- A smooth, user-friendly image slider built with **Swiper.js** makes it easy to explore the resort’s amenities.
- Hovering over each photo reveals a brief description, enhanced with responsive **Tailwind CSS** styling.

✅ **Contact Form**

- A membership sign-up form built using **React Hook Form**, including real-time validation for a smoother user experience.
- Comes with a **short survey** to gather visitor preferences.

✅ **Overall Resort Design**

- Every element — from the logo, color palette, and UI design to the descriptive copy — was thoughtfully created from scratch, with careful attention to detail to make it feel like a real luxury resort
- Lightweight and elegant layout built with **Tailwind CSS**
- Immersive user experience brought to life through smooth animations powered by **Framer Motion**
- **Fully responsive design** optimized for seamless viewing on desktop, tablet, and mobile devices

## What's Improved? 🧐 (2025/03)

**(EN)**

After the initial development, I refined the UI and improved the user experience — **especially on mobile** — through the following refactorings:

- **Unified font styles** to bring a more cohesive and polished look to the design
- **Adjusted font sizes** for mobile to improve readability
- Fixed a bug where the hamburger menu briefly displayed its contents on page load
- Added a **scroll-to-top function** to the logo, making it easier to navigate without opening the mobile menu
- Resolved layout issues in the Facilities Swiper by resizing images and optimizing swipe behavior
- Fixed missing validation messages in the contact form submission

- **Locked dependency versions**
  - Removed carets (`^`) from `package.json` to prevent version drift across environments
  - **Improved build stability and consistency across devices**
    <br />

**(JP)**

初期開発後、UI の統一性やユーザー体験の向上を目的として、特に**モバイル環境における表示**を中心に以下のリファクタリングを実施しました：

- **フォントスタイルを統一**し、全体のデザインに一貫性と洗練さを追加
- モバイル表示時の**フォントサイズを調整**し、可読性を向上
- 読み込み時に一瞬だけハンバーガーメニューの中身が見えていた問題を修正

- Facilities セクションの Swiper に一部崩れがあったため、画像サイズとレイアウトを調整し、スムーズに操作できるよう改善

## Installation

### 🚧 Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/en) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- [Git](https://git-scm.com/)

❗️This project was bootstrapped with [Vite](https://vite.dev/), a lightning-fast frontend build tool.
<br />

### 📦 Key Dependencies

- [Tailwind CSS](https://tailwindcss.com/)
  Utility-first CSS framework used for responsive layout and design customization.
- [Framer Motion](https://motion.dev/)
  Enables smooth animations and transitions for enhanced UX and interactivity.
- [React Hook Form](https://react-hook-form.com/)
  Used to build the contact form with validation and user-friendly form handling.
- [Swiper.js](https://swiperjs.com/)
  Modern slider library for the Facilities section, allowing smooth image swiping.
- [IconsScout](https://iconscout.com/all-assets/dog?price=free) & [Font Awesome](https://fontawesome.com/)
  A combination of scalable vector icons and custom icon sets for consistent UI visuals.
- [Unsplash Images](<(https://unsplash.com/ja/s/%E5%86%99%E7%9C%9F/dog)>)
  High-quality dog and resort images used for non-commercial purposes in UI design.
  <br />

## Usage 🚀

### 1. Clone this repository

```bash
git clone https://github.com/sahoooii/ts-react-dogation
```

### 2. Navigate to the project folder

```bash
cd ts-react-dogation
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. 🔧 Environment Variables

Rename the `example.env` file to `.env` and add the following

```bash
# Rename the file
mv example.env .env

# And add your form endpoint
REACT_APP_FORM_SUBMIT=ADD_YOUR_GET_FORM_INFO
```

## 📘 Development Notes

**(EN)**

This project was originally developed around 2023, when I was becoming more confident with React. To improve maintainability and scalability, I chose to build this project entirely with **TypeScript** from the beginning.

The concept of **Dogation** comes from combining two things I love: my dog and the island of Hawaii, where I once lived. The name blends Dog and Vacation — an idea born from my real-life experiences of staying at pet-friendly resorts.

Every part of the site — from logo, colors, and copywriting to the structure and flow of the facility introduction — was thoughtfully crafted by me. I wanted to create not just a website, but a **memorable experience**, enhanced with natural scroll-triggered animations and a fully working contact form.

This project reflects my passion for someday building meaningful services for dogs and their owners. **My own dog Shokota** even makes a special appearance, so I hope you enjoy exploring the resort with him 🐶🐾
